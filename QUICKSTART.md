# Quick Start Guide

## Installation & Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to `http://localhost:3000`

## Building Your First Website

### Step 1: Add Components
- Look at the **Component Library** on the left sidebar
- **Drag** any component (like Heading, Text, Button) onto the canvas in the center

### Step 2: Customize Components
- **Click** on any component you've added to select it
- The **Property Panel** on the right will show up
- Edit properties like:
  - Text content
  - Colors (background, text)
  - Padding and margins
  - Image URLs
  - Input labels and placeholders

### Step 3: Build Your Layout
- Use **Container** components to group related elements
- Add **Form** components to group inputs together
- Arrange components by dragging them around

### Step 4: Preview Your Website
- Click the **Preview** button in the top right
- See exactly how your website will look
- Click **Exit Preview** to return to editing

### Step 5: Export Your Website
- Click **Export HTML** button
- Your website will download as `website.html`
- Open it in any browser or deploy it anywhere!

## Troubleshooting

**Components not dragging?**
- Make sure you're clicking and holding on the component in the library
- Drag it over the canvas area before releasing

**Can't see Property Panel?**
- Click on a component first to select it
- The Property Panel only appears when a component is selected

**Preview looks different?**
- Make sure you've saved your changes (they auto-save)
- Try refreshing the preview
